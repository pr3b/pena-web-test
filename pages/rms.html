What we did:

SDK documentation

The Company:

Retail Merchant Services (RMS) is one of the largest payment solutions in the UK. They provide a range of card machines and online payment solutions with integration options.

The Approach:

We revamped their SDK documentation to help businesses integrate with their cloud-based software, SmartBridge. We produced SDK documentation, complete with integration and installation guides for their users, using transformative designs. It included the templates, code samples, code, and tools required to connect the EPOS system and the merchant's card machine.

The Results:

Our documentation provided a comprehensive guide on how to connect a merchant's card machine and EPOS using SmartBridge. Pena was able to build the bridge between the merchant’s payment system and EPOS system to bring together every customer's needs into one place using Smartbridge.
