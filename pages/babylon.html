Babylon

Open the docs

What we did:

API Documentation

The Company:

Microsoft’s Babylon (Babylon.js) is a website-based image search engine. It provides users with a real-time 3D engine using a JavaScript library fordisplaying 3D graphics in a web browser via HTML5.

The Approach:

Pena revamped and updated Babylon’s documentation from Babylon’s central concept to practice. We intended to create a more comprehensive document so that users obtain all required information from one source.

The Results:

Babylon now has an all-in-one document comprising Babylon’s core concepts, system components, installation & usage instructions, and API & CLI command documentation. Our involvement has made it the users and developers easier to maximize Babylon utilization, both its features and app integrations.
